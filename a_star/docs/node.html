Used in A* to explore the state space of a sliding tile puzzle.

Parameters:

  • action: The action taken to arrive at state. None if this is a root node.
  • parent: The parent node of this node. None if this is a root node.
  • path_cost: The sum of every step cost from the root to this node.
  • state: A SlidingTileState.
class Node:
# 
    def __init__(self, parent=None, action=None, path_cost=None, state=None):
        self.state = state
        self.action = action
        self.parent = parent 
        self.path_cost = path_cost

    def get_parent(self):

Override the equality operator to easily search for nodes in the explored set and frontier.

    def __eq__(self, other):
        return self.state == other.state
